# SkillShare Platform

A collaborative Skill-Sharing and Learning Platform built with Spring Boot.

### Clone the Repository

git clone https://github.com/Dinusha-Ariyarathna/skillShare.git
cd YOUR-REPO


***Follow this pattern 

main      ← Protected, production-ready
└── dev   ← Shared development branch
    └── feature/your-name-task ← Individual feature branches

***How to Contribute
1. Sync with dev before starting
git checkout dev
git pull origin dev

2. Create a new feature branch
git checkout -b feature/your-name-task

3. Make changes, commit, and push
git add .
git commit -m "Add: your change description"
git push origin feature/your-name-task

4. Create a Pull Request (PR)
Go to GitHub and create a PR from your feature branch → dev

Once approved and tested, it will be merged into dev
